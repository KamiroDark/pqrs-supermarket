# 🛒 Sistema PQRS — SuperMarket

Proyecto académico — Ingeniería de Software I  
Fundación Universitaria Konrad Lorenz · 2026-1

## Descripción

Aplicación para la recepción y gestión de Peticiones, Quejas,
Reclamos y Sugerencias (PQRS) de los clientes de SuperMarket.

**App Web responsiva** — radicación de PQRS (clientes) y gestión
de radicados (gestor PQRS).

## Stack tecnológico

| Capa        | Tecnología              |
|-------------|-------------------------|
| Frontend    | Angular 17+             |
| Backend     | Node.js + Express       |
| Base datos  | MySQL                   |
| ORM         | Sequelize               |
| Integración | API REST                |

## Estructura del proyecto

```
pqrs-supermarket/
├── frontend/       # App Angular (web + responsiva móvil)
├── backend/        # API REST con Node.js + Express
├── base-de-datos/  # Scripts SQL
└── docs/           # Documentación del proyecto
```
